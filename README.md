# دعوة زفاف أحمد وروان

Arabic responsive wedding invitation for Ahmed & Rawan.

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.2.7.

## Development server

To start a local development server, run:

```bash
ng serve
```

Once the server is running, open your browser and navigate to `http://localhost:4200/`. The application will automatically reload whenever you modify any of the source files.

## Building

To build the project run:

```

This will compile your project and store the build artifacts in the `dist/` directory. By default, the production build optimizes your application for performance and speed.

```

```text

```

## Styling with Tailwind

This project uses Tailwind CSS v4 through Angular's PostCSS pipeline.

- Global Tailwind import and wedding theme tokens live in `src/styles.css`.
- PostCSS configuration lives in `.postcssrc.json`.
- Prefer Tailwind utility classes for layout, spacing, colors, typography, and form states.
- Keep custom CSS in `wedding-invitation.component.css` only for bespoke animation, envelope motion, flower rain, and illustration slicing.
